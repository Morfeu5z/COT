# - Tmux -
### For Linux Terminal
#### Program, który pozwala w jednym terminalu odpalić kilka.

<div align="justify">

### Cheet Sheet:
>#### Zacznijmy od tego, że:
>>       ctrl + b + ...
>używane jest do wszystkiego.
>#### Dzielenie okna w poziomie
>>      ctrl + b + %
>#### Dzielenie okna w pionie
>>      ctrl + b + "
>#### Zamknięcie jednego terminala
>>      ctrl + b + c
>#### Przejście po oknach
>>      ctrl + b + strzałki
